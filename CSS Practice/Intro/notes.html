CSS (Cascading Style Sheets)

CSS is a styling language used to control the layout and appearance of web pages. It was introduced in 1994. Each file of CSS is called a sheet.

Ways to Apply CSS

  1. Inline Styling
  2. Internal Styling
  3. External Styling

1. Inline Styling

This method applies styles directly within an element using the style attribute.

<h1 style="color: red; background-color: blue;">Hello World</h1>

Example:

Hello World

2. Internal CSS

This method uses the <style> tag inside the <head> section. You can select elements using selectors and apply styles.

<style>
    h2 {
        color: green;
        background-color: lightyellow;
    }
</style>

Example:

Hello WOrld

3. External CSS

This method uses a separate CSS file with a .css extension. The file is linked to HTML using the <link> tag.

<link rel="stylesheet" href="style.css" />

CSS Selectors

1. Tag Selector

Selects all elements of a specific tag.

h1 {
    color: blue;
}

Example:

Heading using tag selector

2. Class Selector

Select multiple elements with the same class name using class attribute.

HTML:
<p class="myClass">This paragraph uses a class selector.</p>

CSS:
.myClass {
    color: green;
    background-color: lightgrey;
}

Example:

This paragraph uses a class selector.

3. ID Selector

Select a single unique element using id attribute.

HTML:
<p id="myId">This paragraph uses an ID selector.</p>

CSS:
#myId {
    color: purple;
    background-color: yellow;
}
