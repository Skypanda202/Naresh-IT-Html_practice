Colors in CSS

A good design depends on its brand colors. Use minimal colors for user interface to maintain clarity and purpose.

Colors can be added in multiple ways in CSS:

  • Hex Code: #FF0000 → Red color
  • RGB: rgb(255,0,0) → Red color
  • RGBA: rgba(255,0,0,0.5) → Red color with 50% opacity
  • Name: red → Red color

Fonts in CSS

Fonts represent brand identity. Use minimal fonts for consistency.

  • Serif: Has strokes on corners. Example: Times New Roman
  • Sans-serif: No strokes on corners. Example: Arial, Inter
  • Monospace: Same width letters. Example: Courier New
  • Cursive: Curved and styled. Example: Comic Sans MS
  • Fantasy/Modern: Unique decorative fonts. Example: Impact

Font properties:

  • font-weight: bold, normal, 100-900
  • font-style: normal, italic
  • font-size: small, medium, large, x-large, px
  • text-align: left, right, center, justify
CSS Units

Units represent measurements in CSS:

  • px: Pixels, fixed size
  • %: Percentage of parent element
  • vw: Viewport width (percentage of screen width)
  • vh: Viewport height (percentage of screen height)
