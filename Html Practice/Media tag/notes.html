Media in HTML

HTML5 introduces many media elements such as video, audio, iframe, track, and picture. These are used to present media content inside web pages.

1. Video Tag

The <video> tag is used to embed videos on web pages. You can control playback using attributes like controls, autoplay, muted, and loop. The <source> tag provides the video file using the src attribute.

<video height="200" controls autoplay muted loop>
    <source src="PathName.mp4" type="video/mp4">
</video>

2. Audio Tag

The <audio> tag is used to play audio files. It also uses controls, autoplay, and loop attributes. The <source> tag specifies the audio file.

<audio controls autoplay loop>
    <source src="AudioPath.mp3" type="audio/mpeg">
</audio>

3. Iframe Tag

The <iframe> tag is used to display another web page inside your webpage. It can also embed PDFs, Docs, or third-party websites. You can set height, width, and frameborder.

<iframe src="https://www.example.com" width="400" height="200" frameborder="0"></iframe>

4. Track Tag

The <track> tag is used inside videos to add captions or subtitles. It uses attributes like src (file), kind (subtitles/captions), and srclang (language).

<video height="200" controls>
    <source src="sample-video.mp4" type="video/mp4">
    <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
</video>

5. Picture Tag

The <picture> tag is used to display responsive images. Different images can be loaded for different screen sizes using source elements.

<picture>
    
    <img src="default.jpg" alt="Responsive Image" width="300">
</picture>
