HTML Form Validation Notes

required

Makes the input mandatory.

<input type="text" required placeholder="This is required">
readonly

Input cannot be edited but will be submitted.

<input type="text" value="Cannot change" readonly>
disabled

Input cannot be edited and will not be submitted.

<input type="text" value="Disabled field" disabled>
minlength

Minimum number of characters allowed.

<input type="text" minlength="4" placeholder="At least 4 chars">
maxlength

Maximum number of characters allowed.

<input type="text" maxlength="10" placeholder="Max 10 chars">
min

Minimum numerical value allowed.

<input type="number" min="18" placeholder="Min 18">
max

Maximum numerical value allowed.

<input type="number" max="60" placeholder="Max 60">
pattern

Input must match a regular expression (regex).

Pattern Symbols:

  • [] — group
  • { } — range
  • + — at least 1 of the previous value
  • * — at least 0 times of the previous value
  • . — wildcard, matches any character
  • \d — digits (0-9)
  • \w — word characters (letters, digits, underscore)
  • \s — whitespace characters

Example 1: 4 uppercase letters followed by 4 digits

<input type="text" pattern="[A-Z]{4}[0-9]{4}" placeholder="ABCD1234">

Example 2: Valid email format

<input type="email" pattern="[A-Za-z0-9.-_@#]+@[A-Za-z]+\.[A-Za-z]{2,}" placeholder="example@mail.com">
