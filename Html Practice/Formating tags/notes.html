HTML Formatting Tags

These tags are used to format text. Search engine–friendly tags are <strong>, <em>, <ins>, and <del>.

Basic Formatting Tags

<b>Text</b> → This text is bold

<i>Text</i> → This text is italic

<strong>Text</strong> → This text is bold (important)

<em>Text</em> → This text is italic (emphasized)

<u>Text</u> → This text is underlined

<ins>Text</ins> → This text is inserted (underlined)

<s>Text</s> → This text is strikethrough

<del>Text</del> → This text is deleted

x<sup>2</sup> → x2

H<sub>2</sub>O → H2O

Special Text Tags
