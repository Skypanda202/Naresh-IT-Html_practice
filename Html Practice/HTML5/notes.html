HTML5 Features

HTML5 provides new features such as semantic elements, responsive images, SVG, canvas, drag and drop, etc.

SVG (Scalable Vector Graphics)

SVG is used to create diagrams and vector graphics directly in HTML.

Circle:

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" fill="red"></circle>
</svg>

Rectangle:

<svg width="400" height="400">
    <rect x="100" y="100" height="200" width="200" fill="blue"></rect>
</svg>

Custom Shape (Path):

<svg width="200" height="200">
    <path d="M 50 50 L 150 50 L 150 150 L 50 50 Z" fill="green"></path>
</svg>

Explanation of Path Commands:

  • M - Move to the point
  • L - Draw a line to a point
  • Z - End of the path
Canvas

Canvas element is used to draw 2D or 3D graphics using JavaScript.

<canvas id="mycanvas" width="300" height="150"></canvas>
Drag and Drop

HTML5 provides drag and drop functionality using draggable="true" attribute on elements.

Drag Me 
<div draggable="true" style="width:150px; height:50px; background-color:lightcoral;">
    Drag Me
</div>

Drag and drop requires JavaScript to handle drop events.
