Tables in HTML

Tables in HTML are used to represent data in rows and columns.

  • The horizontal sets are called rows.
  • The vertical sets are called columns.

Basic Table Structure

The whole table is represented inside the <table> element.

  • Each row is written inside a <tr> tag.
  • Each column is written inside a <td> (table data) or <th> (table heading) tag.

Example:

<table border="1">
    <tr>
        <th>Name</th>
    </tr>
    <tr>
        <td>Akash</td>
    </tr>
</table>

Output:

Name
Akash

Merging Rows and Columns

To combine rows or columns, we can use rowspan or colspan attributes inside the first element we want to merge. 

<table border="1">
    <tr>
        <th colspan="2">Student Details</th>
    </tr>
    <tr>
        <td rowspan="2">Akash</td>
        <td>Maths</td>
    </tr>
    <tr>
        <td>Science</td>
    </tr>
</table>

Output:

Student Details
Akash Maths
Science

Adding Captions

We can add a title to your table using the <caption> tag. 

<table border="1">
    <caption>Student Marks</caption>
    <tr>
        <th>Name</th>
        <th>Marks</th>
    </tr>
    <tr>
        <td>Akash</td>
        <td>90</td>
    </tr>
</table>

Output:

Student Marks
Name Marks
Akash 90

Semantic Table Elements

To help search engines and screen readers understand the table better, we can use:

  • <thead> - Table header section
  • <tbody> - Table body section
  • <tfoot> - Table footer section
<table border="1">
    <thead>
        <tr>
            <th>Name</th>
            <th>Marks</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Akash</td>
            <td>90</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">End of Results</td>
        </tr>
    </tfoot>
</table>

Output:

Name Marks
Akash 90
End of Results
