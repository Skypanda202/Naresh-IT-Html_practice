Lists in HTML

Lists in HTML are used to list out certain items. Lists are basically three types:

  1. Ordered Lists
  2. Unordered Lists
  3. Definition Lists
1. Ordered List

Items are displayed in a specific order using the <ol> tag:

<ol>
  <li>Raj</li>
  <li>Shekhar</li>
  <li>John</li>
</ol>
  1. Raj
  2. Shekhar
  3. John
2. Unordered List

Items are displayed without order using the <ul> tag:

<ul>
  <li>Raj</li>
  <li>Shekhar</li>
  <li>John</li>
</ul>
  • Raj
  • Shekhar
  • John
Nested Lists

You can place one list inside another:

<ul>
  <li>
    Friends
    <ol>
      <li>Hello</li>
      <li>Bye</li>
    </ol>
  </li>
</ul>
  • Friends
    1. Hello
    2. Bye
3. Definition List

Used for terms and their definitions using <dl>, <dt>, and <dd>:

<dl>
  <dt>HTML</dt>
  <dd>Hypertext Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  <dt>JS</dt>
  <dd>JavaScript</dd>
</dl>

Block vs Inline Elements

Block elements always start on a new line and take up full width:

<div>This is a DIV (block element)</div>
<div>This is another DIV (block element)</div>
This is a DIV (block element)
This is another DIV (block element)

Inline elements stay in the same line and take only as much width as needed:

<span>This is a SPAN (inline)</span>
<span>This is another SPAN (inline)</span>
This is a SPAN (inline) This is another SPAN (inline)
