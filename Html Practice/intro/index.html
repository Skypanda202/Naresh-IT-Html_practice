Web Dev Notes: VS Code & HTML Basics

A simple, HTML-only page you can open directly in any browser.

1) VS Code: Install & Setup

Visual Studio Code (VS Code) is a code editor for writing HTML, CSS, and JavaScript. It has many extensions and a large community.

Install steps:

  1. Go to code.visualstudio.com/download and download the installer for your operating system.
  2. Open the downloaded file, accept the terms, then click Next until installation completes.
  3. Open VS Code → File → Open Folder… and choose your project folder.

2) Basic Settings in VS Code

  • Auto Save: File → Auto Save (enable).
  • Compact Folders: Settings → search “Compact Folders” → uncheck.

3) Extensions to Install

  • Live Server (by Ritwick Dey) — start a local server and auto-refresh your page.
  • Prettier (by Prettier.io) — auto-format your code for a consistent style.
  • Material Theme Icons — nicer file and folder icons in Explorer.

4) Web Technologies Overview

Web technologies power web pages and websites. The modern web began with the World Wide Web (WWW) at CERN by Tim Berners-Lee. Early markup experiments led to HTML (HyperText Markup Language), which runs in all browsers. The current version in common use is HTML5.

5) HTML Basics

Comments

Use comments to explain your code. Browsers ignore them.

<!-- This is a comment --> <!-- Comments start with "<!-- and end with -->" -->

Tags & Elements

HTML uses tags to wrap content. Most tags have an opening and a closing form.

<tagname> ... content ... </tagname>

Attributes

Tags can include attributes to provide extra information.

<h1 title="heading">Some text</h1>

Document Structure

A basic HTML page has two children inside <html>: <head> (metadata like title) and <body> (the visible content).

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

6) Handy Shortcuts (Emmet)

In VS Code, type ! and press Enter to insert an HTML5 template. You can also use html:5 and press Enter. 

!      → expands to an HTML5 boilerplate in VS Code
html:5 → same as above

Indentation

Left-side spacing is called indentation. Good code uses consistent indentation and comments for readability.
