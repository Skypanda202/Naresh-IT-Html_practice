HTML Forms - Complete Notes

1. Introduction to Forms

Forms are used to collect user input and send data between the client and server. Common examples include login, registration, contact, feedback, address, application, and payment forms.

2. Form Structure

Every form is enclosed by the <form> element.

  • action – The URL or file where the submitted data will be sent.
  • method – The HTTP method used to send data. Two common methods:
    • GET – Appends data into the URL.
    • POST – Sends data in the request body (hidden from URL).

3. Basic Form Example

<form action="submit.php" method="post">
    <input type="text" name="username" placeholder="Enter your name">
    <button type="submit">Submit</button>
</form>

4. Input Attributes

  • type – Specifies the input type (text, password, email, number, etc.).
  • id – Unique identifier for linking with labels.
  • name – Key used to send data to server.
  • placeholder – Shows hint inside input field.

5. Using Labels

<label for="username">Username:</label>
<input type="text" id="username" name="username">
